# NVIDIA + Jupyter + Deep Learning

## What's Included

* CUDA 9.0 (Ubuntu 16.04 base)
* Python data science packages
  * numpy, scipy, etc.
  * matplotlib is pre-cached
  * Extras also included: NLTK, OpenCV, Scikit-Learn
* TensorFlow and Keras
* PyTorch
